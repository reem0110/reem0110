- **Key Features:**
  - Interactive Google Maps interface displaying real-time user location

**3. Obstacle Detection System for Visually Impaired (Arduino + Companion Mobile App)**

- **Objective:** Design a low-cost assistive device to enhance mobility and safety for blind individuals through real-time obstacle detection.
- **Key Features:**
  - Ultrasonic sensors (HC-SR04) for detecting obstacles in the user's path
  - Buzzer alerts integrated into footwear for immediate haptic/audio feedback
  - Bluetooth connectivity to companion mobile app for additional monitoring
- **Technical Stack:** Arduino (C/C++), HC-SR04 Ultrasonic Sensors, Buzzer/Vibration Motors, Bluetooth Module, Flutter Companion App
- **Results & Achievements:**
  - Achieved reliable obstacle detection within a 4-meter range
  - Response time under 200ms for immediate hazard alerts
  - Demonstrated stable performance through rigorous indoor/outdoor testing
- **Evidence:** Photos of hardware assembly, circuit diagrams, and app interface available in README
